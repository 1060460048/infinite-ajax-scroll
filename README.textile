h1. Infinite Ajax Scroll

Turn your paginated pages into infinite scrolling pages with ease.

Version 0.1.0

Requires jQuery 1.4 or newer.

h2. Installation

To install Infinite Ajax Scroll make sure you have jQuery 1.4+ installed. Next add jquery.ias.js to the head of your webpage.

Installation of Uniform is quite simple. First, make sure you have jQuery 1.4+ installed. Then youâ€™ll want to link to the jquery.uniform.js file and uniform.default.css in the head area of your page:

h2. Basic usage

jQuery.ias({
	container : ".listing",
	item: ".post",
	pagination: "#content .navigation",
	next: ".next-posts a",
	loader: "images/loader.gif"
});
